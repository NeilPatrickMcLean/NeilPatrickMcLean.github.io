## Introduction

This website serves as my personal portfolio, showcasing web design projects completed throughout my coursework and career. It highlights my experience using HTML5 and CSS3 to create responsive, accessible, and visually appealing websites while demonstrating the skills I have developed in modern web development. The purpose of this portfolio is to provide potential employers, clients, and collaborators with examples of my work and the technical abilities I have gained.

## Features

- **Responsive layout**: Adapts to different screen sizes and devices for optimal viewing.
- **User-friendly navigation**: Consistent menu structure for easy access to all sections.
- **Portfolio section**: Showcases completed web design projects with visual examples.
- **About Me section**: Provides background information and highlights skills.
- **Contact section**: Allows visitors to reach out for professional inquiries.
- **Clean, modern design**: Built using HTML5 and CSS3 for a contemporary look.
- **Accessibility**: Organized page structure that follows web development best practices to ensure usability for all users.

## Technologies Used

## Installation

To set up this project locally on your machine:

### Prerequisites

* A modern web browser (Chrome, Edge, Firefox, or Safari)
* Git (optional, if cloning the repository)

### Clone the Repository

```bash
git clone https://github.com/NeilPatrickMcLean/NeilPatrickMcLean.github.io
cd NeilPatrickMcLean/NeilPatrickMcLean.github.io
```

### Run the Website

Open the `index.html` file in your preferred web browser.

## Usage

Use the navigation menu to browse the different sections of the portfolio, including the About Me, Projects, and Contact pages. The website is responsive and can be viewed on desktop, tablet, and mobile devices.
